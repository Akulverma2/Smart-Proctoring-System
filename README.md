# Smart-Proctoring-System
AI-based real-time exam monitoring system using computer vision and object detection to detect suspicious activities like multiple faces, phone usage, and head movement, with logging and dashboard visualization
#  Smart Proctoring System
## Features
- Face Detection
- Eye Tracking
- Head Movement Detection
- Phone Detection (YOLOv8)
- Real-time Alerts
- Logging System
- Dashboard Visualization

## Tech Stack
- Python
- OpenCV
- YOLOv8
- Streamlit

## How to Run
```bash
python proctor.py
python -m streamlit run dashboard.py
